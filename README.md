# pokemon-cli

> Play Pokemon in CLI

install your CLI locally:

```sh
$ [sudo] npm link
```

Run:

```sh
$ pokemon-cli
```

```sh
$ pokemon-cli -x
```

```sh
$ pokemon-cli -y
```

Run unit tests with mocha:

```sh
$ npm test
```

Publish:

```sh
$ npm publish
```
